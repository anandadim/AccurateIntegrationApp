## METODE 1: Mulai Ulang (Paling Mudah & Bersih)
*Gunakan cara ini jika Anda bekerja sendirian dan tidak keberatan kehilangan riwayat commit (history) sebelumnya.*

1.  **Hapus Folder Git Lokal**
    * Buka folder proyek di laptop Anda.
    * Hapus folder tersembunyi bernama `.git`.
2.  **Hapus Repository di GitHub**
    * Buka GitHub > Settings > General > Scroll ke bawah > **Delete this repository**.
3.  **Buat Repository Baru**
    * Buat repo baru dengan nama yang sama.
4.  **Upload Ulang (Tanpa .env)**
    Buka terminal di folder proyek laptop Anda:
    ```bash
    # 1. Pastikan .env sudah masuk .gitignore
    echo ".env" >> .gitignore

    # 2. Inisialisasi ulang
    git init
    git add .
    git commit -m "Initial commit (Clean)"
    git branch -M main

    # 3. Sambungkan ke repo baru
    git remote add origin [https://github.com/USERNAME_ANDA/NAMA_REPO.git](https://github.com/USERNAME_ANDA/NAMA_REPO.git)
    git push -u origin main
    ```
